# Security Policy

## Supported Versions

Currently, only the latest version of Sidebark is supported for security updates.

## Reporting a Vulnerability

If you discover a potential security vulnerability in Sidebark, please do NOT create a public issue. Instead, please follow these steps:

1. Go to the [Security](https://github.com/phine-apps/sidebark/security) tab of this repository.
2. Click on **Advisories**.
3. Click **Report a vulnerability** to privately disclose the issue.

We will acknowledge your report and provide an estimated timeframe for a fix.
