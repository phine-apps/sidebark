## How Can I Contribute?

### Reporting Bugs

This section guides you through submitting a bug report for Sidebark. Following these guidelines helps the maintainers (phine-apps) and the community understand your report, reproduce the behavior, and find related reports.

- **Check the existing issues** to see if the bug has already been reported.
- **Use the Bug Report template** when opening a new issue.
- **Provide a clear description** of the bug and steps to reproduce it.

### Suggesting Enhancements

- **Check the existing issues** to see if the feature has already been requested.
- **Use the Feature Request template** when opening a new issue.

### Pull Requests

- **Follow the existing code style.**
- **Include tests** if you're fixing a bug or adding a feature.
- **Ensure the build passes.**

## Development Setup

1. **Clone the repository.**
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Perform initial build:**
   ```bash
   npm run build
   ```
4. **Load the extension in your browser:**
   - Go to `chrome://extensions`.
   - Enable **"Developer mode"**.
   - Click **"Load unpacked"** and select the `dist` folder.
